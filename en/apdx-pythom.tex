\noindent
The following na√Øve algorithm was used to compute the energies. Better performance could be achieved in many ways, for example by using the Newton's method to find the solutions of $F(...) = 0$ and the Levenberg-Marquardt algorithm to find the minimum.

\lstinputlisting[language=Python]{../num/robin-lowest-energy.py}
\begin{lstlisting}[language=Python]

# set up limits and steps
step_e = 0.1
step_p = 0.01
step_a = 0.1
min_a = -2
max_a = 2

estimates = [
    (a, e := find_minimum_energy(a, e, step_p, step_e))
    for a in float_range(min_a, max_a, step_a)
]

print_estimates(estimates)
\end{lstlisting}
